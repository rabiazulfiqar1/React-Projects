# React Projects

A collection of React-based projects demonstrating various concepts and features of React development, from basic fundamentals to complete web applications.

## ğŸ“ Projects Overview

This repository contains three main projects:

### 1. **Chatbot Project** 
A modern, interactive chatbot application built with React and Tailwind CSS.

**Key Features:**
- Real-time chat interface
- Message persistence using localStorage
- Custom chatbot responses
- Modern UI with Tailwind CSS
- Component-based architecture

**Technologies Used:**
- React 19.1.0
- Vite
- Tailwind CSS 4.1.12
- SuperSimpleDev library
- Day.js for date/time handling

### 2. **E-commerce Project**
A complete e-commerce web application with multiple pages and routing functionality.

**Key Features:**
- Product browsing homepage
- Shopping cart and checkout system
- Order management
- Package tracking
- Multi-page routing with React Router

**Technologies Used:**
- React 19.1.0
- React Router 7.8.0
- Vite
- Component-based architecture

### 3. **React Basics**
Foundational React examples and exercises demonstrating core React concepts.

**Includes:**
- Basic React setup with CDN
- JSX syntax examples
- Component rendering
- Interactive elements (counter, chatbot basics)
- Display-as-you-type functionality

